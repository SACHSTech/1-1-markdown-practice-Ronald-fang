# Tricky Examples
Test your knowledge of Markdown syntax with these examples. Can you re-create them without looking anything up?

## Example 1
1. Programming Languages
- Python
  - a. Popular Frameworks 
       - Django
       - Flask
  - b. Notable Features
    - Dynamic typing
    - Huge ecosystem
- JavaScript
  - a. Frontend
    - React
    - Vue
    - Angular
  - b. Backend
    - Node.js
    - Deno

2. Databases
- Relational
  - a. PostgreSQL
  - bMySQL
- NoSQL
  - MongoDB
  - Redis
